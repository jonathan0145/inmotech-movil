# Pantalla Detalle de Propiedad visual

Se replica el diseño visual de la pantalla de detalle de propiedad del frontend web en Flutter, usando los mismos colores, fuentes y estructura.

- Imagen destacada de la propiedad
- Título, dirección, precio
- Descripción y características
- Botón para contactar o hacer oferta
- Estilos similares a la web
