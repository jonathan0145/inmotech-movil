# Lista de Pages de la App Móvil

Esta es la lista de todas las páginas principales implementadas o requeridas en la app móvil, siguiendo la estructura y lógica del frontend web y la documentación visual.

---

## Pages implementadas en `lib/pages/`

- **login_page.dart** — Página de inicio de sesión
- **register_page.dart** — Página de registro de usuario
- **dashboard_page.dart** — Dashboard principal (según rol)
- **profile_page.dart** — Perfil de usuario
- **properties_page.dart** — Listado de propiedades
- **property_detail_page.dart** — Detalle de propiedad
- **chat_page.dart** — Chat entre usuarios
- **notifications_page.dart** — Centro de notificaciones
- **offers_page.dart** — Gestión de ofertas
- **main_navigation.dart** — Navegación principal
