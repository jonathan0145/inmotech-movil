# Pantalla Chat visual

Se replica el diseño visual de la pantalla de chat del frontend web en Flutter, usando los mismos colores, fuentes y estructura.

- Listado de mensajes tipo burbuja
- Campo de texto para escribir mensaje
- Botón de enviar
- Estilos similares a la web
