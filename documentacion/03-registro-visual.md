# Pantalla Registro visual

Se replica el diseño visual de la pantalla de registro del frontend web en Flutter, usando los mismos colores, fuentes y estructura.

- Formulario con campos de usuario, correo y contraseña
- Botón de registro
- Enlace para volver a login
- Estilos similares a la web
