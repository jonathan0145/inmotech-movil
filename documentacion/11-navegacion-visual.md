# Navegación visual entre pantallas

Se implementa una barra de navegación inferior moderna para moverse entre las pantallas principales de la app:

- Dashboard
- Propiedades
- Chat
- Ofertas
- Notificaciones
- Perfil

Cada ícono lleva a la pantalla correspondiente, replicando la experiencia de la web pero adaptada a móvil.
