# Pantalla Login visual

Se replica el diseño visual de la pantalla de inicio de sesión del frontend web en Flutter, usando los mismos colores, fuentes y estructura.

- Formulario con campos de usuario y contraseña
- Botón de inicio de sesión
- Enlace para registro
- Estilos similares a la web
